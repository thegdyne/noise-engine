NOISE ENGINE - macOS QUICK START
=================================

INSTALL
-------

1. SuperCollider
   Download from: https://supercollider.github.io/downloads
   Drag to Applications

2. Python 3.11+
   Download from: https://www.python.org/downloads/
   Tick "Add Python to PATH" during install

3. Get the code
   Download ZIP from: https://github.com/thegdyne/noise-engine
   (Green "Code" button → Download ZIP)
   Extract to Desktop

4. Set up Python (in Terminal)

   cd ~/Desktop/noise-engine-main
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt

   If that fails, install manually:
   python3 -m pip install PyQt5
   python3 -m pip install python-osc
   python3 -m pip install python-rtmidi
   python3 -m pip install pyyaml
   python3 -m pip install numpy
   python3 -m pip install Pillow
   python3 -m pip install scipy
   python3 -m pip install librosa

5. Install presets

   python3 tools/forge_gen_preset.py --all --install

   IMPORTANT: Type "--all --install" by hand!
   Chat apps turn -- into — which breaks the command.


RUN
---

Step 1: Start SuperCollider
   Open SuperCollider
   File → Open → noise-engine-main/supercollider/init.scd
   Cmd+A (select all)
   Cmd+Enter (run)
   Wait for "Noise Engine ready!" in the console

Step 2: Start the GUI
   cd ~/Desktop/noise-engine-main
   source venv/bin/activate
   python3 src/main.py

Step 3: Connect
   Click "Connect to SuperCollider" (top right)
   Green = connected

Step 4: Load a preset
   Click Load → select "harga"
   Turn up Master Volume carefully!


PANIC CONTROLS
--------------

If it gets too loud:
   • Mute buttons on channel strips
   • Master Volume fader
   • In SuperCollider: type s.freeAll; then Cmd+Enter
   • Quit SuperCollider entirely


QUICK CONTROLS
--------------

Cmd+M          Modulation Matrix
Cmd+X          Cross Matrix

Generator controls:
   Top row      Custom params (unique per generator)
   Bottom       Freq, Cutoff, Res, Attack, Decay
   ENV/CLK      2nd button - switch envelope mode
   Filter       Top-right button - cycle filter types

Channel strips: Volume, Pan, EQ, Mute/Solo
Master: Global EQ, Limiter, Master Volume


TROUBLESHOOTING
---------------

"python not found"
   Use python3 instead of python

"No module named PyQt5" (or similar)
   python3 -m pip install PyQt5
   (or whichever module is missing)

"Unrecognized arguments —all"
   Type the flags manually, don't copy-paste

"No such file or directory"
   Make sure you're in noise-engine-main:
   cd ~/Desktop/noise-engine-main

"No presets"
   Run the preset install again with both flags together:
   python3 tools/forge_gen_preset.py --all --install

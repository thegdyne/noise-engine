◎ Modulation Sources

Noise Engine has a dedicated modulation system with 4 slots, each providing 4 outputs. This gives you 16 independent modulation buses that can be routed to any parameter in any generator.

Two modulation source types are available: LFO (clock-synced with quadrature phase outputs) and Sloth (Lorenz-based chaos for slowly evolving textures).

Bus Architecture

Each mod slot has 4 outputs. Bus index = (slot - 1) × 4 + output

Any mod slot can host any generator type. The slot/output configuration is flexible — you could run 4 LFOs, 4 Sloths, or any combination.

Sloth — Chaos Generator

Lorenz Attractor

Sloth implements a Lorenz chaotic system — the same mathematics that describes atmospheric convection and the famous "butterfly effect". The result is slowly-evolving, never-repeating modulation that adds organic movement to your patches.

Inspired by the Nonlinearcircuits Triple Sloth — a Eurorack module based on cascaded chaotic circuits. Our digital implementation captures the same slowly-drifting character.

Four Outputs

Each Sloth provides 4 outputs derived from the Lorenz system:

X

Lorenz X axis
Primary chaos — most active, widest range

Y

Lorenz Y axis
Secondary chaos — related to X but decorrelated

Z

Lorenz Z axis
Tertiary chaos — slowest, most subtle movement

R

Rectified gate
Positive-only signal — triggers when X crosses zero

X, Y, and Z are mathematically related but never identical — perfect for creating "similar but different" modulation across multiple parameters. R provides a pseudo-random gate signal derived from the chaos.

Output Invert (NORM / INV)

Per-Output Signal Flip

Each output has an INVERT button that flips the signal polarity. This creates contrary motion between outputs even when they share the same waveform and phase.

NORM (default)
INV (inverted)

Use case: Route LFO outputs A and B to left and right pan. Set B to INV. Now when A pans left, B pans right — instant stereo animation.

For Sloth, inverting X while keeping Y normal creates "anti-correlated" modulation — when one parameter rises, the other falls, but both follow the same chaotic trajectory.

Two-Process Architecture

Python (PyQt5) handles all UI, state management, and user interaction.
SuperCollider does all audio synthesis and DSP.
They communicate via OSC messages over localhost.

Signal Path Details

1. Generators

  • 8 independent synthesis slots
  • 22 generator types available
  • Each has filter + envelope
  • Stereo output per slot

2. Channel Strips

  • Volume fader (0 to +6dB)
  • Stereo pan control
  • 3-band parametric EQ
  • Mute / Solo per channel

3. Master Bus

  • DJ-style 3-band isolator
  • SSL G-Series bus compressor
  • Brick-wall limiter @ -0.3dB
  • Master volume control

4. Output

  • Stereo main output
  • Peak + RMS metering
  • Gain reduction display
  • Device selection

Modulation Layer

Mod Sources

  • 4 mod slots
  • LFO (8 waveforms, quadrature)
  • Sloth (chaos generator)
  • 16 mod buses total

Mod Destinations

  • 40 destinations per generator
  • All 5 standard params
  • All 5 custom params
  • Up to 4 sources per dest

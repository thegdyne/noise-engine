8-slot noise generator with modular architecture

Run different noise synths simultaneously. Each generator has 5 standard params plus up to 5 custom params, multimode filter (LP/BP/HP), and clock-synced envelope.

3 LFOs with sync-to-clock. 4-slot FX chain. Central BPM control.

Python frontend talks to SuperCollider backend via OSC. Adding a new generator = one .scd file + one .json config.

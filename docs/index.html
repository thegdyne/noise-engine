What is it?

A texture instrument for drones, ambience, and evolving soundscapes

Synthesis

  • 30 generators across 5 families
  • Custom parameters (P1-P5) per generator
  • Transpose ±2 octaves, portamento
  • 6 filter modes: LP, HP, BP, Notch, LP2, OFF
  • Clock-synced envelopes (13 divisions)

Modulation

  • 4 slots × 4 outputs = 16 mod buses
  • LFO: SIN/SAW/SQR/S&H + 6 phase presets
  • Sloth chaos: Torpor/Apathy/Inertia
  • Full routing matrix with amount/offset
  • Real-time scope visualization

Mixer

  • 8 channel strips with 3-band EQ
  • Fader, pan, mute, solo per channel
  • Echo and Reverb sends
  • Master EQ, compressor, limiter
  • Lo/Hi cut filters per channel

Performance

  • MIDI input with per-slot channel
  • Trigger modes: OFF / CLK / MIDI
  • Keyboard mode for computer input
  • BPM control 20-300 with tap tempo
  • Full preset save/load

Installation

Get started in minutes

Python
3.11+
SuperCollider
3.13+ (3.14 recommended)
Platforms
macOS · Linux · Windows

macOS

✓ Tested on Sonoma/Ventura
brew install --cask supercollider
brew install python@3.11
git clone https://github.com/thegdyne/noise-engine.git
cd noise-engine
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python main.py Full macOS guide →

Linux (Fedora)

✓ Tested on Fedora 39/40

NOISE_ENGINE

Modular noise synthesizer • Python GUI • SuperCollider audio

Python SuperCollider PyQt5 SSOT 95%

8-slot noise generator with modular architecture

Run different noise synths simultaneously. Each generator has multimode filter (LP/BP/HP), clock-synced envelope, and independent parameters.

3 LFOs with sync-to-clock. 4-slot FX chain. Central BPM control.

Python frontend talks to SuperCollider backend via OSC. Adding a new generator = one .scd file + one line of config.

Current Features

◈ Generators

  • 8 independent slots
  • Test Synth, PT2399-based
  • Per-gen: FRQ, CUT, RES, ATK, DEC
  • Multimode filter (LP/BP/HP)
  • Clock-synced envelope
  • 8 clock rates (x8 to /16)

◎ Modulation

  • 3 LFOs
  • Waveforms: SIN, SAW, SQR, S&H
  • Free running or clock sync
  • Separate clock division per LFO

◐ Effects

  • 4-slot serial FX chain
  • Fidelity (bit crush / sample rate)
  • Per-effect amount control

▥ Mixer

  • 8 channel faders
  • Mute / Solo per channel
  • Master output
  • Audio/MIDI status

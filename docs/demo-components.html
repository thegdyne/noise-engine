The Five Parameters

FRQ

Frequency / Rate

Base pitch or primary oscillator frequency. Exponential curve for musical control.

20Hz → 8kHz

CUT

Filter Cutoff

Multimode filter cutoff frequency. Works with LP, HP, and BP modes.

20Hz → 16kHz

RES

Filter Resonance

Filter Q / emphasis at cutoff. Inverted control (up = more resonant).

0.1 → 1.0

ATK

VCA Attack

Envelope attack time. How quickly the sound rises.

0.1ms → 2s

DEC

VCA Decay

Envelope decay/release time. How long the sound rings out.

10ms → 10s

Multimode Filter

Every generator has a state-variable filter with three modes, controlled by CUT and RES.

LP

Low Pass - removes highs

HP

High Pass - removes lows

BP

Band Pass - isolated band

Clock-Synced Envelope

ATK and DEC control an AD envelope on the VCA. The envelope can be triggered by:

OFF

Drone mode - continuous sound

CLK

Triggered by master clock

MIDI

Triggered by MIDI notes

Clock division rates:

/32 /16 /12 /8 /4 /2 CLK x2 x4 x8 x12 x16 x32

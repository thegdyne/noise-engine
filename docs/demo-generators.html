One File = One Generator

Adding a new generator is simple: create one .scd file (SuperCollider SynthDef) and one .json file (config). That's it. The system auto-discovers and loads them.

supercollider/generators/ ├── subtractive.scd ← SynthDef code ├── subtractive.json ← UI config + custom params ├── karplus.scd ├── karplus.json ├── granular.scd ├── granular.json └── ...

Standard + Custom Parameters

Every generator gets 5 standard params (FRQ, CUT, RES, ATK, DEC) that work identically across all types. Then each generator can define up to 5 custom params (P1-P5) for type-specific controls.

Example: Karplus has custom params for DAMP (string damping) and RTG (retrigger rate). Granular has SIZE (grain size) and DENS (grain density).

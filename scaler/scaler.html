‚ö†Ô∏è Critical Build Instructions

  • Track Cuts First: Make 3 track cuts BEFORE soldering any components (marked with red X on layout)
  • IC Socket: Use socket for IC1 - solder socket first, insert IC last
  • Bypass Caps: C1 and C2 MUST be within 0.5" of IC pins 7 and 4 (prevents oscillation)
  • Component Orientation:
    • IC notch points toward row A (top)
    • Electrolytic C5: positive (+) leg toward output (right)
    • LED: long leg (+) toward +12V (row A), short leg toward GND via R6
  • Power Connections: Double-check ¬±12V polarity BEFORE powering on
  • Trimpot Access: Position RV1 with adjustment screw accessible from board edge
  • Test Points: Before connecting to MOTU/synth, verify:
    • Pin 7 (IC) = +12V
    • Pin 4 (IC) = -12V
    • Output with 0V input = 0-50mV
  • Calibration: Use multimeter in DC voltage mode. Set MOTU output to 3.600V, adjust RV1 until CV output reads exactly 10.000V
